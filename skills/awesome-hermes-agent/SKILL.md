## Getting Started (3 Steps)

1. **Get running** — [Official Docs quickstart](https://hermes-agent.nousresearch.com/docs/)
2. **Add skills** — Install [wondelai/skills](https://github.com/wondelai/skills) (380+ stars)
3. **Get a GUI** — [hermes-workspace](https://github.com/outsourc-e/hermes-workspace) or [mission-control](https://github.com/builderz-labs/mission-control)
